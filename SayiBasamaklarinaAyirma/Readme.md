## Zengin Sayılar


### Tanım
##### Bir sayının bölenleri toplamı kendisinden büyük olan sayılardır.



### Örnek
##### 2018 sayısı için,
- Birler basamagi  : 8
- Onlar basamagi   : 1
- Yuzler basamagi  : 0
- Binler basamagi  : 2
